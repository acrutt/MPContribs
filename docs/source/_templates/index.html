Materials Project (MP) aims to establish itself as a hub for datasets that are contributed from several different researchers, and directly integrated under a common platform. To this end, the MPContribs Framework enables users to add their own data to MP, and use the underlying tools to interact with and reference that data.
MPContribsUsers is a collaborative extension to the MPContribs core. It contains reusable routines for pre-submission processing (raw data organization and MPFile generation) and post-submission processing (analyses integrated with MP core data and Web UIs). It allows for the collaborative involvement of the framework developer while simultaneously establishing an archive of example files and scripts to support future contributors and to reduce duplication of work.
